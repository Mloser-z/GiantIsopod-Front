# GiantIsopod_Front
大学三年级第三学期小组实训项目的前端。项目包括Vue源码以及docker部署相关文件。

## 项目结构

```
./
├── docker-compose.yaml
├── nginx/
├── README.md
└── web/
    ├── babel.config.js
    ├── docker-compose.yaml
    ├── Dockerfile
    ├── jsconfig.json
    ├── package.json
    ├── package-lock.json
    ├── public/
    ├── README.md
    ├── src/
    │   ├── apis/
    │   ├── App.vue
    │   ├── assets/
    │   ├── components/
    │   ├── main.js
    │   └── views/
    └── vue.config.js


```
nginx文件夹存放nginx的Docker部署相关配置，web界面的Vue源码以及配置在web文件夹下存放。</br>
